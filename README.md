## Setup Instructions

### 1. Install dependencies for the main Electron project
```sh
npm i
```

### 2. Run all Electron services
```sh
npm run serve:all
```

### 3. Install dependencies for electron-app subproject
Navigate to the `electron-app` directory:
```sh
cd electron-app
npm i
```

### 4. Start the electron-app
```sh
npm run start
```
